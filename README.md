## Overview

This project demonstrates how to build a dynamic Kanban Board in Oracle APEX using JavaScript, AJAX callbacks, and SortableJS.

A Kanban board is a visual workflow management tool that helps track tasks across different stages such as:

* To Do
* In Progress
* Done

This implementation provides an interactive user interface with drag-and-drop functionality and real-time updates.

## Features

* Drag and drop task movement
* Real-time status updates using AJAX
* Dynamic rendering using JavaScript
* Visual workflow tracking
* Clean and responsive user interface
* Easy integration with Oracle database

## Technologies Used

* Oracle APEX
* JavaScript
* SortableJS (Drag and Drop Library)
* PL/SQL (AJAX Callback Processes)
* HTML and CSS
